# Python Assignment — Part 2 (Restaurant Order System)

## Overview

In this assignment, I worked with Python dictionaries and lists to build a simple restaurant system.
The idea was to understand how nested data works and how to process it step by step.

---

## Tasks Covered

### Task 1 — Explore the Menu

* Printed menu category-wise
* Counted total items and available items
* Found the most expensive item
* Listed items below ₹150

---

### Task 2 — Cart Operations

* Added items to cart with quantity
* Updated quantity if item already exists
* Handled invalid and unavailable items
* Removed items from cart
* Generated final order summary with total and GST

---

### Task 3 — Inventory Tracker

* Created a deep copy of inventory
* Checked that original data is not affected
* Updated stock based on cart items
* Displayed reorder alerts when stock is low

---

### Task 4 — Sales Log Analysis

* Calculated total revenue per day
* Found the best-performing day
* Found most ordered item
* Added a new day and recalculated values
* Printed all orders using numbering

---

## Concepts Used

* dictionaries and nested dictionaries
* lists and loops
* conditionals
* basic string handling
* deepcopy from copy module
